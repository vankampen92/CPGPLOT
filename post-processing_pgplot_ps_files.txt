Command history:

1. Selecting the interesting page (2nd page, in this case) and storing it in BASIN.ps:
 
>$ psselect -p2 pgplot.ps BASIN.ps

2. Rewriting the ps file with the "right" borders. 

>$ ps2ps BASIN.ps BASIN_GOOD.ps

3. Converting the ps file to eps file, with the "right" bounding box

>$ ps2eps BASIN_GOOD.eps

In addition, look at the web site, for more information on compressing 
the resulting figure if it results too large. 

http://www.mrao.cam.ac.uk/~bn204/technotes/pgplotastroph.html
